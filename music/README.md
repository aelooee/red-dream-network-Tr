# 背景音乐说明

## 使用方法

1. 请将您想要作为背景音乐的MP3文件放在此文件夹中
2. 将文件重命名为 `background-music.mp3`
3. 刷新网页，背景音乐将尝试自动播放

## 音乐建议

为了获得最佳体验，建议选择：

- 古典中国风音乐，如古筝、琵琶等传统乐器演奏的曲目
- 音量适中、节奏舒缓的曲目
- 循环播放效果好、无明显起伏的背景音乐
- 文件大小建议控制在5MB以内，以保证加载速度

## 音量控制

网页右下角提供了音乐控制面板，您可以：

- 点击播放/暂停按钮控制音乐播放状态
- 点击音量图标快速静音/取消静音
- 使用音量滑块调节背景音乐音量大小

## 注意事项

- 系统会尝试在页面加载后自动播放背景音乐，但由于浏览器安全策略限制，可能需要用户首次与页面交互后才能播放
- 如果音乐没有自动播放，请点击页面任意位置或使用播放按钮手动开始播放
- 请确保您使用的音乐拥有合法的使用权限
